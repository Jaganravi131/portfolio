# Portfolio Website

A production-ready Next.js portfolio foundation for a creator who works across design, branding, illustration, and development.

## Direction

This scaffold is built around one unified narrative:

- demonstrate craft through visual work,
- prove technical depth through shipped projects,
- support business growth through services and contact,
- create room for articles, talks, and thought leadership later.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content model

The first version is designed to be easy to expand with:

- featured projects,
- services,
- selected writing,
- testimonials,
- a stronger contact flow,
- future case studies.

Replace placeholder information with your real bio, projects, links, and contact details.
